 Site Oficial | KrakenLabs
Website futurista com temática cyberpunk, combinando design neon avançado com interações dinâmicas para destacar serviços de tecnologia e segurança digital.

Visão Geral
O site incorpora:
Design Cyberpunk Neon (Ciano #00FFFF, Rosa #FF00FF, Verde #00FF00)
Animações Avançadas (scroll, fade-in, efeitos de observação)
Interatividade Fluida (scroll suave, triggers de elementos)
Performance Otimizada (IntersectionObserver para carregamento eficiente)

Destaques Técnicos
JavaScript Cyberpunk
Feature	Descrição
Scroll Suave	Navegação entre seções com animação fluid (behavior: 'smooth')
Observer System	Anima elementos conforme aparecem na tela (opacity + transform)
Staggered Loading	Efeito de aparecimento em cascata para textos (h1-h6, p, li, etc)
Performance	Uso inteligente de IntersectionObserver para ativar animações sob demanda.
